## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)

### Setup
- Clone the repository ` git clone https://github.com/niicode/webpack-todo-app.git`

### Install
- `npm install`
- `npm run build`
- `npm start`
